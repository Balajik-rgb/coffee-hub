📌 Project Description
CoffeeHub is designed as a modern and elegant café website that focuses on user experience and visual storytelling. The goal of this project was to create a real-world website layout that a coffee shop could actually use to showcase its brand, products, and customer engagement.

The design emphasizes simplicity, readability, and responsiveness, ensuring users get a smooth experience across devices.

---

🌟 Core Features & Sections

🔹 Navigation Bar
A clean and responsive navigation bar that allows users to easily access different sections like Home, Menu, About, Offers, Testimonials, and Contact.

🔹 Hero Section
A visually appealing landing section with a strong headline and call-to-action button. It creates the first impression and draws users into the website experience.

🔹 Menu Section
Displays coffee items such as Espresso, Latte, and Cappuccino with images, descriptions, and pricing. Designed using a grid layout for clarity and aesthetics.

🔹 About Section
Tells the story of the CoffeeHub brand, focusing on quality, ambiance, and customer connection.

🔹 Special Offers Section
Highlights promotional deals like “Buy 1 Get 1 Free” and Happy Hour discounts to improve customer engagement.

🔹 Testimonials Section
Includes real-style customer feedback to build trust and authenticity.

🔹 Contact Section
A simple and effective contact form allowing users to send messages directly (UI level implementation).

🔹 Footer Section
Contains copyright details and social media icons for extended connectivity.

---

🛠️ Technologies Used

- HTML5 – For structuring the web content using semantic elements
- CSS3 – For styling, layout design, and responsiveness
- Google Fonts – For elegant typography (Playfair Display & Roboto)
- Responsive Design Techniques – Ensuring compatibility across mobile, tablet, and desktop devices
